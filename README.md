# Movie Review Sentiment Analysis
 This research project, as a final project for Florida Polytechnic University's Data and Text Mining course, seeks to address how the degradation of Intellectual Property can occur when poorly receives fims are released. By understanding those that engage with movies, we can gain unique insights into evaluating film outcomes over time.

 Note: There is a fourth dataset that this project utilized, but its contents were too large for GitHub to store. This dataset is called rotten_tomatoes_critic_reviews. This file can be located and downloaded here: https://www.kaggle.com/datasets/stefanoleone992/rotten-tomatoes-movies-and-critic-reviews-dataset
